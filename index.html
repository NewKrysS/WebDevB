Pokemon Speedruns - Generation 1

This website has been created as a small front-end project using guides from the Firebase youtube channel. Only HTML5, CSS3 and Javascript has been used for this mini-project.

It serves as a mockup guide to start speedrunning pokemon games of first generation, this includes:
